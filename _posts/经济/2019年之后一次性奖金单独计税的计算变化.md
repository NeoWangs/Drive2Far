2019年的个税改革，居民个人在取得全年一次性奖金后，年终奖扣税可选择单独计税或并入综合所得计税。单独计税这种方式保留了下来，而且比 2011 年老版优惠的多。下面讲一下单独计税这种方式的逻辑，以及与 2011 年之前的计算结果差异。


单独计税的核心公式很简单：

```
应纳税额 = 全年一次性奖金收入 * 适用税率 - 速算扣除数
```

但是适用税率不是直接看年终奖总额，而是先把年终奖除以12个月，再去找“按月换算后的综合所得税率表”。也就是：

```
月换算数 = 年终奖 / 12
```

比如年终奖14.4万元，月换算数就是：144000 / 12 = 12000元。

在2019版税率表里，12000元仍然落在10%这一档，所以应纳税额为：

```
144000 * 10% - 210 = 14190元
```

到手就是：

```
144000 - 14190 = 129810元
```

注意：速算扣除数不乘以12。它只是用“除以12”后的数额找税率和速算扣除数，真正计算时仍然拿全年奖金总额乘税率，再减一次速算扣除数。

总结：
```
先算 avg = bonus / 12
按 avg 查月度税率和速算扣除数
tax = bonus * rate - deduction
afterTax = bonus - tax
```

我们把2019版单独计税用到的月度税率表列一下（图形可看：https://show.cssass.com/bonus.html ）：

| 月换算数 | 税率 | 速算扣除数 | 对应年终奖区间 |
| --- | ---: | ---: | ---: |
| 不超过3000元 | 3% | 0 | 不超过3.6万 |
| 超过3000元至12000元 | 10% | 210 | 3.6万至14.4万 |
| 超过12000元至25000元 | 20% | 1410 | 14.4万至30万 |
| 超过25000元至35000元 | 25% | 2660 | 30万至42万 |
| 超过35000元至55000元 | 30% | 4410 | 42万至66万 |
| 超过55000元至80000元 | 35% | 7160 | 66万至96万 |
| 超过80000元 | 45% | 15160 | 超过96万 |

再看2011版旧表。2011版的税率还是3%、10%、20%、25%、30%、35%、45%这几档，但是前面几档的区间窄得多：

| 月换算数 | 税率 | 速算扣除数 | 对应年终奖区间 |
| --- | ---: | ---: | ---: |
| 不超过1500元 | 3% | 0 | 不超过1.8万 |
| 超过1500元至4500元 | 10% | 105 | 1.8万至5.4万 |
| 超过4500元至9000元 | 20% | 555 | 5.4万至10.8万 |
| 超过9000元至35000元 | 25% | 1005 | 10.8万至42万 |
| 超过35000元至55000元 | 30% | 2755 | 42万至66万 |
| 超过55000元至80000元 | 35% | 5505 | 66万至96万 |
| 超过80000元 | 45% | 13505 | 超过96万 |


差距主要来自前几档区间被拉宽了。还是以14.4万元年终奖为例：

2019版：

```
144000 / 12 = 12000
税率10%，速算扣除数210
税额 = 144000 * 10% - 210 = 14190元
```

2011版：

```
144000 / 12 = 12000
税率25%，速算扣除数1005
税额 = 144000 * 25% - 1005 = 34995元
```

两者相差：

```
34995 - 14190 = 20805元
```

也就是说，在14.4万元这个点上，2019版比2011版少交约2.08万元税。这个数字之所以这么显眼，是因为14.4万元刚好站在2019版10%档的最高处，但在2011版里已经进入25%档。

再看几个金额：

| 年终奖 | 2019版税额 | 2011版税额 | 少交税 |
| ---: | ---: | ---: | ---: |
| 3.6万 | 1080 | 3495 | 2415 |
| 10.8万 | 10590 | 21045 | 10455 |
| 14.4万 | 14190 | 34995 | 20805 |
| 30万 | 58590 | 73995 | 15405 |
| 30万零1元 | 72340.25 | 73995.25 | 1655 |

30万这个位置也挺有意思。30万元刚好还在2019版20%档内，所以和2011版差距仍然不小。一旦超过30万元，2019版也进入25%档，和2011版的税率就基本拉齐了，后面大部分差距主要只剩速算扣除数的差额，也就是1655元。

总结：2019版对30 万以下的年终奖非常友好，尤其是3.6万、14.4万、30万这些新档口附近；年终奖超过30万以后，和2011版相比差距就不大了。

不过单独计税还有一个反直觉的问题：它并不是对全年奖金分段累进，而是拿全年奖金总额套同一个税率。结果就是，一旦刚好跨过档口，多发1元，税可能突然多出一大截。

比如14.4万元时：

```
税额 = 144000 * 10% - 210 = 14190元
到手 = 129810元
```

如果多发1元，变成144001元，就进入20%档：

```
税额 = 144001 * 20% - 1410 = 27390.2元
到手 = 116610.8元
```

奖金多了1元，到手反而少了13199.2元。这个不是算错了，而是单独计税公式本身造成的税率跳跃。年终奖发放如果能规划，最需要避开的就是这些“多发少拿”的档口。
